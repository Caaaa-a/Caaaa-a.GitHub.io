注意⚠️：

  1. torch，torchvision，torchaudio和python版本需要对应
  2. 安装的torch要与gpu对应（可以向下兼容）
  3. 反复安装要重启能解决一些问题

下载速度太慢

添加镜像源（清华、科大），在官网下载时去掉命令后面的 -c pytorch

下载对应的whl文件（也可以先下好）

在清华源里找对应的bz2（也可以先下好）

详见：conda安装Pytorch下载过慢解决办法(11月26日更新ubuntu下pytorch1.3安装方法)

torch，torchvision，torchaudio和python版本需要对应

报错：from torch._C import ImportError:*torch/undefined symbol: Py│Slice_Unpack

详见： ImportError: /home/gp/………/torch/lib/libtorch_python.so: undefined symbol: PySlice_Unpack

各版本对应关系：官网

安装的torch要与gpu、算力对应（可以向下兼容）

CUDA error: no kernel image is available for execution on the device

解决办法：安装老版本pytorch

算力对应版本详见：pytorch 报错 RuntimeError: CUDA error: no kernel image is available for execution on the device
