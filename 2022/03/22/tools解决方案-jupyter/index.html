自动补全&代码格式化

  1. 安装 nbextensions 安装：pip install jupyter_contrib_nbextensions -i https://pypi.mirrors.ustc.edu.cn/simple 激活：jupyter contrib nbextension install --user

  2. 安装 nbextensions_configurator 安装：pip install jupyter_nbextensions_configurator 激活：jupyter nbextensions_configurator enable --user

  3. 启动 jupyter notebook 点击新增加的Nbextensions项

  4. 勾选Hinterland（代码提示/自动补全）和Autopep8（代码格式化）
